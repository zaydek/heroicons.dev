## Get Started (Vite)

Start the development server:

```sh
npm run dev
# Or npx vite dev
```

Build the production-ready build:

```sh
npm run build
# Or npx tsc && npx vite build
```

Serve the production-ready build:

```sh
npm run preview
# Or npx vite preview
```
